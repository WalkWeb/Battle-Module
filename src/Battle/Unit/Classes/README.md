
# Планы по классам

- Люди
    - [x] Паладин `Paladin` {танк}
        - Сокрушающий удар (Crushing Blow) — Паладин делает мощный удар, наносящий урон магией жизни
        - Оглушающий удар (Stunning Strike) — Выбрав подходящий момент, паладин делает неожиданный удар, который 
        оглушает противника
        - Авангард (Vanguard) — Призывая силы света на помощь, паладин восстанавливает свое здоровья, и получает бонус 
        ко всем сопротивляемостям
    - [x] Убийца `Assassin` {дд ближнего боя} — особенность: Шанс критического удара -20%, сила критического удара +100%
        - Кровоточащая рана — Наносит глубокий порез противнику, из-за которого он будет постепенно терять здоровье
        - Повреждение рук — Выбрав подходящий момент убийца делает удар по рукам противника, снижая наносимый им урон
        - Удар в спину — Делает ловкий рывок за спину противника, нанося ему смертельный удар в самое уязвимое место
    - [x] Рейнджер `Ranger` {дд дальнего боя}
        - Взрывной выстрел — Выстреливает стрелой с взрывчатым веществом, за счет которого стрела наносит большой урон 
        огнем и поджигает противника
        - Хорошая позиция — Рейнджер занимает наилучшую позицию на поле боя, увеличивая свою меткость и защиту
        - Град стрел — Делает несколько очень быстрых выстрелов один за одним
    - [x] Маг Стихий `Elemental Mage` {маг}
        - Огненный шар — Концентрирует стихию огня, превращая её в огненный шар, и обрушивает его на противника
        - Воспламенение — Поджигает противника, который начинает получать большой урон огнем в течении небольшого 
        количества времени
        - Метеоритный дождь — Обрушивает на противников метеоритный дождь, который повреждает одну или несколько целей 
        до трех раз
    - [x] Жрец `Priest` {поддержка} — особенность: Здоровье -10%, Мана +10%
        - Осколки льда — Создает и обрушивает на противника ледяные осколки
        - Поддержка воды — Накладывает баф, постепенно восстанавливающий здоровье
        - Исцеление — Восстанавливает значительное количество здоровья самому раненому члену отряда
- Эльфы
    - [x] Хранитель `Templar` {танк}
        - Разящий клинок — Делает разящий удар клинком с увеличенным шансом и силой критического удара
        - Круговая оборона — Хранитель занимает оборонительную боевую позицию, увеличивая на некоторое время шанс блока 
        атак и заклинаний
        - Крепость — Хранитель накладывает на себя баф, увеличивая сопротивления и немного максимальные сопротивления на 
        несколько ходов
    - [x] Дневной охотник `Day Hunter` {дд дальнего боя} — особенность: Защита +20%, Магическая защита -20%
        - Отравляющее жало — Охотники знают, какими травами нужно обработать лезвие оружия, чтобы оно не только нанесло
         урон, но и отравляло противника
        - Поддержка ветра — Призывая силы ветра себе на помощь, охотник увеличивает скорость своих движений, получая 
        бонус к скорости атаки и защите
        - Пронзающий удар — Сконцентрировав всю свою ярость охотник делает сокрушительный удар, который наносит огромный 
        урон и имеет увеличенный шанс нанести критический удар
    - [x] Ночной охотник `Night Hunter` {дд дальнего боя} — особенность: Защита -20%, Магическая защита +20%
        - Ледяной выстрел — Превращает стрелу в ледяной осколок, которая нанесет большой урон магией воды
        - Паучья сеть — Призывает из тьмы паутину, которая обволакивает и сковывает противника, снижая его скорость 
        атаки, скорость создания заклинаний и меткость
        - Залп тьмы — Наносит сокрушительный урон магией тьмы. 25% от нанесенного урона воруется в здоровье
    - [x] Заклинатель `Caster` {маг}
        - Шаровая молния — Создает шаровую молнию, которая наносит урон магией воздуха
        - Единство с ветром — Незначительно увеличивает скорость атаки, создания заклинаний, меткость и защиту на 
        несколько ходов
        - Вихрь — Создает мощный вихрь, который проходит по врагам, нанося урон магией воздуха два раза подряд
    - [x] Оракул `Oracle` {поддержка}
        - Осколки льда — Концентрирует влагу из воздуха, которая сразу замерзает, и обрушивается на противника ледяными
         осколками, нанося урон водой
        - Восстановление — Накладывает баф на себя или союзника, постепенно восстанавливающий здоровье
        - Решимость — Накладывает комплексное усиление на себя или союзника, увеличивает урон, меткость и защиту
- Орки
    - [x] Титан `Titan` {танк} — особенность: Здоровье +20%, Мана -50%
        - Удар ногой — Наносит удар ногой, который сбивает врага с ног. Сбитый с ног противник теряет возможность 
        совершать действия
        - Регенерация — Раскрывает резервные силы, которые постепенно восстанавливают небольшое количество здоровья
        - Защитная стойка — Титан принимает защитную стойку, увеличивая свои сопротивления
    - [x] Разрушитель `Destroyer` {дд ближнего боя}
        - Гнев — Вложив весь свой гнев, разрушитель делает сильный удар, наносящий увеличенный урон
        - Грязный прием — Разрушитель делает удар по кистям противника, уменьшая скорость атаки и защиту цели на 
        несколько ходов
        - Боевой клич — Когда бой затянулся разрушитель использует боевой клич, увеличивающий свой и без того не малый 
        урон на несколько ходов
    - [x] Берсерк `Berserk` {дд ближнего боя} — особенность: Регенерация здоровья +2%, Регенерация маны -2%, Расход 
    выносливости -1
        - Град ударов — Делает три быстрых удара подряд
        - Боевой транс — Берсерк входит в боевой транс, в котором его обычная и магическая защита вырастает
        - Ликантропия — Чем дольше продолжается бой, тем сильнее берсерк превращается в разъяренного зверя, в форме 
        которого его скорость атаки значительно вырастает
    - [x] Шаман `Shaman` {поддержка}
        - Ядовитый дух — Призывает ядовитый дух, который обрушивается на противника, нанося ему урон магией смерти
        - Укрепляющее снадобье — Накладывает постепенное восстановление здоровья на самого раненого члена отряда
        - Боевое лечение — Мощное лечение самого раненого члена отряда
    - [x] Боевой маг `Battle Mage` {маг}
        - Сгусток плазмы — Создает раскаленный шар плазмы, и метает его во врага
        - Лечение — Лечит самого раненого члена отряда
        - Метеорит — Обрушивает на врага большой метеорит, который наносит урон огнем

        - Зелье оздоровления — Алхимик дает самому раненому союзнику зелье оздоровления, которое будет постепенно

        - Зелье мощи — Усиливает одного из союзников зельем мощи, которое увеличивает его урон и сопротивления
    - [x] Отшельник `Hermit` {саммонер}
